# Auto-Encoding Variational Bayes

**Authors:** Diederik P. Kingma, Max Welling
**Paper:** arXiv:1312.6114 (2013)

---

## Core Contributions

1. **Stochastic Gradient Variational Bayes (SGVB)** – a differentiable, unbiased estimator of the variational lower bound that works with continuous latent variables under mild differentiability conditions.
2. **Auto-Encoding VB (AEVB)** – an efficient inference and learning algorithm for i.i.d. datasets with per-datapoint continuous latent variables, using the SGVB estimator to jointly optimize a recognition model (encoder) and generative model (decoder).

---

## Problem Statement

- **Data:** i.i.d. dataset X = {x^(i)} generated by: z^(i) ~ p_θ*(z), then x^(i) ~ p_θ*(x|z).
- **Challenges:**
  - Marginal likelihood p_θ(x) = ∫ p_θ(z)p_θ(x|z) dz intractable.
  - True posterior p_θ(z|x) intractable.
  - Mean-field VB integrals also intractable.
  - Large datasets require minibatch optimization.

**Three goals:**
1. Efficient approximate ML/MAP for parameters θ.
2. Efficient approximate posterior inference for latent z given x.
3. Efficient approximate marginal inference for x (e.g., denoising, inpainting).

## Method

### Variational Lower Bound

For each datapoint:

log p_θ(x^(i)) = D_KL(q_φ(z|x^(i)) || p_θ(z|x^(i))) + L(θ,φ; x^(i))

Since KL ≥ 0, L is the lower bound:

L(θ,φ; x^(i)) = -D_KL(q_φ(z|x^(i)) || p_θ(z)) + E_q_φ[log p_θ(x^(i)|z)]

### Reparameterization Trick

Express random variable z as a deterministic variable z = g_φ(ε, x), where ε is an auxiliary variable with independent marginal p(ε), and g_φ(.) is some vector-valued function parameterized by φ.

Then:
E_q_φ(z|x)[f(z)] = E_p(ε)[f(g_φ(ε, x))] ≈ (1/L) Σ f(g_φ(ε^(l), x))

**Example (univariate Gaussian):** z = μ + σε, ε ~ N(0,1).

### SGVB Estimators

**Generic estimator L^A:**
L^A(θ,φ; x^(i)) = (1/L) Σ [log p_θ(x^(i),z^(i,l)) - log q_φ(z^(i,l)|x^(i))]

**Lower-variance estimator L^B** (when KL term is analytic):
L^B(θ,φ; x^(i)) = -D_KL(q_φ(z|x^(i)) || p_θ(z)) + (1/L) Σ log p_θ(x^(i)|z^(i,l))

## Variational Auto-Encoder (VAE) Example

- **Prior:** p_θ(z) = N(z; 0, I)
- **Approximate posterior:** q_φ(z|x^(i)) = N(z; μ^(i), σ^(2,i) I) where μ^(i), σ^(i) are outputs of an MLP encoder.
- **Likelihood:** Gaussian (real data) or Bernoulli (binary data) computed from z via an MLP decoder.
- **Reparameterization:** z^(i,l) = μ^(i) + σ^(i) ⊙ ε^(l), ε^(l) ~ N(0, I).

**KL term analytic solution (Gaussian case):**
-D_KL(q_φ(z|x^(i)) || p_θ(z)) = (1/2) Σ_j (1 + log(σ_j^2) - μ_j^2 - σ_j^2)

## AEVB Algorithm

```
Initialize θ, φ
repeat
    X_M ← random minibatch of M datapoints
    ε ← random samples from p(ε)
    g ← ∇_{θ,φ} L^M(θ,φ; X_M, ε)
    Update θ, φ using gradients (e.g., SGD or Adagrad)
until convergence
return θ, φ
```

*Settings in experiments: M=100, L=1.*

## Significance

The VAE paper introduced the reparameterization trick for training latent variable models with stochastic gradient descent. VAEs became a cornerstone of generative modeling alongside GANs, influencing countless applications in representation learning, semi-supervised learning, and generative AI.
